### 기능 3 : 게시글 입력 및 확인
- write.html : jinja 일부 활용한 게시판 기능 구현 -> 지니뮤직 쿼리 일부 숫자부분 접근(크롤링 발생) -> url api에 입력 후 검색 -> 글제목+내용+별점 입력 후 함께 mongoDB에 데이터 저장 
- 게시판 작성 -> 게시판 작성 확인
- view.html : write에서 접근했던 데이터 내용 앨범아트, 앨범 및 가수, 글제목, 내용 순으로 정렬 후 나열
